## Sobre o Projeto
Este repositório contém os códigos, experimentos e resultados desenvolvidos durante o projeto de Iniciação Científica focado em Otimização Combinatória e Algoritmos Heurísticos.

O objetivo principal é estudar e desenvolver um método de solução baseado na metaheurística de recozimento simulado para resolver o problema de corte de itens irregulares em faixa bidimensional, comparando a eficiência de diferentes abordagens heurísticas em relação a métodos exatos.

## Como executar 

### Via VSCode:
1. Clone o repositório:

`git clone https://github.com/ubiratanpaniago/ic-otimizacao.git`

2. Abra o VSCode com a pasta do projeto

3. Abra o terminal e instale as dependencias:

`pip install pulp`

4. Navegue até a pasta dos códigos:

`cd Códigos`

5. Execute o script:

`python [nome do código]`

### Via Google Colab:
Copie o código e cole no Notebook. 
Obs.: o `pli 2.py` precisa acessar o arquivo da instância, então é necessario fazer upload da instância de interesse no Google Colab.
